APRENDIENDO GIT Y GIT HUB

  1. PRIMER COMANDO : git init (para inicializarlo, una vez por proyecto)
  2. SEGUNDO COMANDO : git add . (para guardar la carpeta que va a ir a la nube )
  3. TERCER COMANDO: git commit -m "mensajito"
  4. CUARTO COMANDO: git status (para conocer el status de mi codigo)
